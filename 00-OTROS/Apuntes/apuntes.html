Navegación básica

Cuando tengamos una página web con distinas páginas, hacer carpetas para las sub páginas, y poner un "index.html" en esta carpeta, para asi obtener una mejor navegación

Ejemplo:

Web con menú (Home, Blog, Contacto)

Para Blog y Contacto, crearemos una carpeta y dentro de esta crearemos un "index.html", asi al navegar en la URL, no nos aparecerá "www.web-ruth/contacto.html"
